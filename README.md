# 📌 README - Repositório de M1-FarmaAura

## 📖 Descrição
Repositório destinado ao registro do trabalho M1 CRUD Farmácia. Foi criado com o objetivo de registrar e acompanhar o desenvolvimento dos trabalhos realizados ao longo do curso.

Aqui são armazenadas todas as etapas do processo, permitindo visualizar a evolução do projeto, desde a ideia inicial até a versão final.


## 🎯 Objetivo

O principal intuito deste repositório é:

- Documentar o progresso do trabalho
- Demonstrar o desenvolvimento de cada etapa  
- Permitir que o professor acompanhe o processo completo  
- Identificar a contribuição de cada integrante da equipe

## 🔄 Processo de Desenvolvimento

O desenvolvimento dos trabalhos segue as seguintes etapas:

1. Planejamento  
2. Pesquisa  
3. Desenvolvimento  
4. Testes  
5. Ajustes finais  

Cada etapa pode ser acompanhada através dos commits realizados no repositório.



## 📌 Controle de Versões

Todas as alterações são registradas por meio de commits, permitindo:

- Acompanhar mudanças no projeto  
- Identificar quem fez cada modificação  
- Visualizar a evolução do trabalho ao longo do tempo  



## 📅 Atualizações

O repositório é atualizado constantemente conforme o andamento dos trabalhos.



## 📎 Observações

Este repositório tem caráter acadêmico e foi desenvolvido com fins de aprendizagem, permitindo transparência no processo de criação dos projetos.
